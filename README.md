# Componentes de una Computadora

## Hardware

### CPU (Unidad Central de Procesamiento)
Es el componente principal encargado de ejecutar instrucciones y procesar datos. 

**Principales Componentes:**
1. **Núcleos (cores)**: Cada núcleo puede manejar tareas de manera independiente.
2. **Caché**: Memoria de alta velocidad que almacena datos temporales para un acceso rápido.

### RAM (Memoria de Acceso Aleatorio)
Es una memoria volátil que guarda datos temporales y programas en ejecución.

**Tipos de RAM:**
- DDR, DDR2, DDR3, DDR4, DDR5

### Almacenamiento
- **HDD (Disco Duro)**: Ofrece mayor capacidad pero menor velocidad comparado con los SSD.
- **SSD (Unidad de Estado Sólido)**: Almacenamiento rápido basado en memoria flash.

### Placa Base (Motherboard)
Placa de circuito principal que conecta todos los componentes de la computadora.

**Componentes Integrados:**
- **Chipset**: Administra la comunicación entre la CPU, RAM y otros componentes.
- **Conectores y Puertos**: Para dispositivos de entrada/salida y almacenamiento.
- **Slots de Expansión**: Para tarjetas adicionales como GPU o tarjetas de sonido.

### Dispositivos de Entrada y Salida
- **Entrada**: Teclado, ratón, escáner, micrófono.
- **Salida**: Monitor, impresora, altavoces.

### Buses
- **Bus de Datos**: Transporta datos entre los componentes.
- **Bus de Dirección**: Transporta las direcciones de memoria de los datos.

## Software

### Sistema Operativo
Gestiona los recursos del hardware y proporciona servicios a las aplicaciones.

**Ejemplos:**
- Windows, macOS, Linux

### Aplicaciones de Usuario
Programas diseñados para tareas específicas del usuario.

**Ejemplos:**
- Microsoft Office, navegadores web, juegos, aplicaciones de diseño gráfico.

### Drivers
Permiten la comunicación entre el sistema operativo y el hardware.

**Ejemplos:**
- Drivers de impresora, tarjeta gráfica.

### Software de Desarrollo
Herramientas utilizadas para crear y mantener software.

**Ejemplos:**
- IDEs como Visual Studio, Eclipse; lenguajes de programación como Python, Java.

# Respuestas a Preguntas

## 1. ¿Cuál es la función principal de cada componente investigado?
- **CPU**: Ejecuta instrucciones y procesa datos.
- **RAM**: Almacena datos temporales y programas en ejecución.
- **Almacenamiento**: Guarda datos permanentemente (HDD y SSD).
- **Placa Base**: Conecta y permite la comunicación entre componentes.
- **Dispositivos de Entrada y Salida**: Permiten la interacción con la computadora.
- **Buses**: Transportan datos y direcciones entre los componentes.

## 2. ¿Cómo interactúa el hardware con el software, cuál es la función de cada uno?
- **Hardware**: Ejecuta instrucciones del software y realiza tareas físicas.
- **Software**: Indica al hardware qué hacer, realiza tareas específicas y ofrece una interfaz al usuario.

## 3. ¿Cuál es la función de la CPU y cuáles son sus partes más importantes?
- **Función de la CPU**: Ejecuta instrucciones y procesa datos.
- **Partes más importantes**:
    - Núcleos (cores)
    - Caché
    - ALU (Unidad Aritmético-Lógica)
    - FPU (Unidad de Punto Flotante)

## 4. ¿Qué es la velocidad de la CPU, también conocida como velocidad del reloj?
- **Velocidad de la CPU**: Mide la rapidez con la que la CPU ejecuta instrucciones, en hercios (Hz).

## 5. ¿Cuál es la secuencia de pasos que ocurre, desde el momento en que presionas el botón de encendido de la computadora, hasta que se muestra el sistema operativo listo para funcionar?
1. Presionar el botón de encendido.
2. Power-On Self Test (POST).
3. Carga del BIOS/UEFI.
4. Detección y configuración del hardware.
5. Carga del bootloader.
6. Inicio del sistema operativo.
7. Carga de servicios y controladores.
8. Interfaz de usuario lista.

## 6. Comenta algo que no sabías y que descubriste en esta actividad.
La complejidad del proceso de arranque de una computadora, desde la secuencia de pasos involucrados hasta la interacción crucial entre hardware y software, fue un descubrimiento interesante y revelador.
